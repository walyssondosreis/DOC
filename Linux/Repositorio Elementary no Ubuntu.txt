A loja de aplicativos do Elementary OS tem diversos aplicativos interessantes, feitos por terceiros e de certa forma exclusivos. No entanto, é possível instalar e rodar facilmente esses aplicativos no Ubuntu 18.04 e no Ubuntu 20.04, bem como em outras distribuições baseadas nessas versões.

A primeira coisa a fazer é adicionar o repositório dos aplicativos do Elementary OS, faça isso executando o seguinte comando em um terminal:

 sudo bash -c 'echo "deb http://packages.elementary.io/appcenter bionic main" >> /etc/apt/sources.list.d/appcenter.list'

O segundo passo é importar a chave GPG do repositório, para isso execute o comando em um terminal:

 sudo wget -O /etc/apt/trusted.gpg.d/appcenter.asc http://packages.elementary.io/key.asc

Na sequência, atualize os repositórios usando o comando abaixo e então terá os novos pacotes disponíveis para instalar.

 sudo apt update

Para remover o repositório, caso não deseje-o mais em seu sistema, use o comando:

 sudo rm -r /etc/apt/sources.list.d/appcenter.list
